## Git 名詞
> * ```Remote``` = 雲端、```Local``` = 自己的電腦
> * ```Repository``` : 資料夾、資料庫
> * ```Clone``` = 複製雲端上的檔案到自己的電腦上
> * ```Commit``` = 存檔點
> * ```Push``` = 推送 commit 到電腦上
> * ```Pull``` = 拉取雲端上的 commit 到自己的電腦上
